# User Journey Maps

## Primary Journey: The Angry Message

### Entry Point
User is triggered, wants to send an angry text/email/message.

### Steps
1. **Discover/Access Vault**
   - Via sidebar navigation
   - Or crisis banner â†’ vault suggestion

2. **Capture the Message**
   - Write freely, no editing
   - Capture the raw emotion

3. **Lock It**
   - Automatic 24-hour timer starts
   - Visual feedback: message "locked"

4. **The Wait**
   - Timer countdown visible
   - Suggestions for what to do instead

5. **Review (24h later)**
   - Re-read with fresh eyes
   - Options: Send, Discard, Keep

### Emotional States
- **Start**: Anger, frustration, urge to act
- **During**: Relief from venting, hope
- **After**: Clarity, often reduced urge to send

## Secondary Journey: The Foggy Mind

### Entry Point
User feels confused, overwhelmed, can't see clearly.

### Steps
1. **Enter The Mirror**
   - Choose flow: "I'm confused" or free chat

2. **Socratic Dialogue**
   - Questions guide self-reflection
   - No advice, just prompts

3. **Insight Moment**
   - User sees something new
   - Aha moment triggers

4. **Capture or Act**
   - Quick capture the insight
   - Or proceed to Decision Journal

### Emotional States
- **Start**: Confusion, overwhelm, stuck
- **During**: Engagement, curiosity
- **End**: Clarity, often surprise at own answer

---

## Tertiary Journey: Emotional Check-in

### Entry Point
User wants to track patterns or understand themselves better.

### Steps
1. **Open Emotional Weather Map**
   - Current state selection
   - Optional intensity and notes

2. **Body Compass (optional)**
   - Map physical sensations
   - Connect body to emotion

3. **Review Patterns**
   - View history
   - See trends over time

### Emotional States
- **Start**: Curious, self-investigating
- **During**: Mindful, present
- **End**: Self-understanding, data for future reference

---

## Crisis Journey

### Entry Point
User expresses self-harm ideation or crisis language.

### Immediate Response
1. **Circuit Breaker Triggered**
   - Conversation pauses
   - Crisis banner appears

2. **Directive Mode**
   - Clear, calm instructions
   - Resource list surfaced

3. **Human Connection**
   - Encourage reaching out
   - No judgment, no delay

### Emotional States
- **Start**: Crisis, pain, isolation
- **During**: Seen, supported
- **End**: Connected to resources, not alone

## Edge Cases

### First-time User
- Landing page explains value proposition
- Quick tour available
- No friction to first feature use

### API Key User
- Settings allows key entry
- AI features unlock immediately
- Clear what data is sent externally

### Offline User
- All static flows work
- Local storage active
- Sync when online

### Returning After Time Away
- Data preserved
- Timer continuations (24h from original lock)
- Welcome back, no pressure
